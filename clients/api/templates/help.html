3 磁盘管理
3.1 添加磁盘
角色和平台:
角色 平台
storage, merge, merge-e pbdata, generic
请求地址:
POST : http://<ip:port>/api/<version>/instances/disks
请求参数:
名称 类型 是否必须 描述
dev_name string 是 磁盘路径, 1.标准机型:使用ces地址; 2.通用机型:使用/dev设备名称
partition_count uint32 否 初始化时, 分区数量, 未指定情况下, 默认为1个分区
disk_type string 否 磁盘类型, 支持hdd/ssd, 当为通用机型的时候, 需要使用该字段标示磁盘类型; 或添加flash卡的时候, 使用该字段标示为SSD类型
响应参数:
名称 类型 是否必须 描述
rc ResponseCode 是 业务逻辑请求返回状态
head Head 是 业务逻辑请求返回头部
请求示例:
curl -i http://127.0.0.1:9000/api/v1.0/instances/disks -X POST  
    -H "Content-Type: application/json" 
    -d '{"dev_name":"1:1:0", "partition_count":2}'
响应示例:
HTTP/1.0 200 OK 
Content-Type: application/json; charset=utf-8
